# nqbtc

[![CI](https://github.com/yjl9903/nqbt/actions/workflows/ci.yml/badge.svg)](https://github.com/yjl9903/nqbt/actions/workflows/ci.yml)

## Usage

```bash
npx nqbtc mcp --base-url=http://localhost:9091/api/v2 --username xxx --password yyy
```
